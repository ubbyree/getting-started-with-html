World Wide Web

WWW is short for the World Wide Web. It is also commonly reffered to as "the web". The web is a collection of interlinked documents contained on numerous computers across the world. A collection of documents on the web are refered to as a web page and are accessed by a "web browser" via the internet. When a web address is typed in a web browser a request is made by that computer to the server at the web address via HTTP. The server then locates the requested HTML document and returns it to the requesting computer. The HTML document is viewed via the web brower that requested the document.



HTML

HTML Stands for Hypertext Markup Language.

HTML is made up of:
Text Content (what you see)
Markup (what it looks like)
References to other documents (ex: images & videos)
Links to other pages



MARKUP / TAGS & ELEMENTS

Markup is a code for formating the layout and style of text by the aplication of numerous tags. Tags create elements and a collection of elements create a HTML document. Most tags have an opeing, contents, and a closing. If you forget to close a tag then it will be applied to all text past it.

Some common tags are:
"b" which makes contents bold
"em" which makes contents italics
"a" (anchor) for making links

The exception to this rule are void tags. Void tags do not contain contents so they don't require closing.

Some common void tag are:
"br" which ends a line of text.
"img" defines an image

Some elements are inline and some are block. Block elements create a box that can have a width and a height.



ATTRIBUTES

Attributes specify a value for the contents of a tag.

Some common attributes are:
"href" which is used with the anchor tag to specify the location of the created link. The contents of the tag will be the name of the link which will be viewable to the user.
"src" is used with the img tag to specify the url of the image to be used.
